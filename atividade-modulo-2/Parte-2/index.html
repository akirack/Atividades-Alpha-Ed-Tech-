Algumas tags html e suas funções

Essa pagina tem como finalidade abordar um pouco sobre tags html e como elas funcionam.

html
É a tag que marca o inicio e o fim da pagina html dizendo a navegador que tudo dentro da pagina deve ser visto como informações html
head
É a tag de cabeçalho onde ficam a configurações da pagina e outros tipos de informações especiais
meta charset="utf-8"
Meta é uma tag para utilização de metadados, que nesse caso diz ao navegador qual o tipo de codificação de texto essa pagina está.
title
É utilizada para da um titulo a pagina. Nesse caso a pagina recebeu o titulo de: Tags html
body
É a tag que marca o inicio e o fim do corpo da pagina html que é onde fica o conteudo.
h1

Todo o texto dentro desse retangulo utliza a tag <h1> utlizada para titulos assim como a primeira linha dessa pagina.

h2

Todo o texto dentro desse retangulo utliza a tag <h2> utlizada para subtitulos assim como a segunda linha dessa pagina.

h3

Todo o texto dentro desse retangulo utliza a tag <h3> que assim como as anteriores são utilizadas em cabeçalhos.

h4

Por fim o texto dentro desse retangulo utliza a tag <h4> que também serve para cabeçalhos.

p

A tag <p> marca os paragrafos como pode ser visto neste texto aqui.

br
A tag <h4> tem como função quebrar o texto.

a
A tag <a> tem como finalidade definir um hiperlink como este: LINK
img

Para colocar uma imagem na pagina, basta utilzar a tag <img>.
